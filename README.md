Un soir, j'étais en appel avec un ami et nous voulions regarder la même vidéo ensemble. Le problème, impossible de la regarder sans perdre en qualité.

J'ai donc créé ce site web permettant à plusieurs personnes de regarder une même vidéo ensemble sans perte de résolution. Le principe est simple une première personne créer un salon et donne le code accès qui a été géner aux autres utilisateurs pour qu'ils se connectent.

En plus de pouvoir regarder la vidéo ensemble, j'y ai inclus quelques fonctionnalités :
* Un chat écrit pour pouvoir réagir en direct.
* La synchronisation, si l'une des deux personnes met en pause, la pause s'active également chez tous les autres utilisateurs, ce qui permet de toujours être synchronisés. De même, lorsqu'elle relance la lecture, celle-ci se déclenche chez tout le monde.
* La possibilité de changer de vidéo sans changer de salon, tout en conservant l'historique des vidéos lues auparavant.
